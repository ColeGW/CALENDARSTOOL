Calendar Tool
Use the tools to build a representation of the season you experience throughout the year. Scroll through the calendar to create an event at any time of the year.
Click for a Tutorial Video
Choose a color to represent a season or event of significance.
Use the Bar tool to add seasonal blocks. Click where you want to create one, this will open a text box for notes and comments. You can resize the bar to the length you want.
You can use the Text Box tool to add more text to the calendar. Each text box can be resized by dragging the lower-right corner.
You can use the Erase tool to remove any item from the calendar.
Try to think deeply about what seasons you experience. As well as the four "classic" seasons, you might experience weather seasons ("mud season," "monsoon season," "fire season"), cultural seasons ("holiday season," "football season," "pumpkin spice season," "McRib season"), industry seasons ("growing season," "busy season"), or countless others that we have yet to think of. Try to share as much as you can!
